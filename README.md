# My_ls

Консольное приложение, которое реализует команду ls в командной строке linux

! Приложение реализует только некоторые параметры:
  - "-l" - выводит подробную таблицу файлов в директории, отсортированных по названию
  - "-r" - изменяет порядок вывода файлов на обратный
  - "-h" - изменяет формат вывода размеров файлов

> [!WARNING]
>
> #### Программа работает только на Linux

## ℹ️Краткие описания файлов

### Consol_myls

    - Консольное приложение, в котором реализована команда ls (все реализованные команды выводятся при запуске)
    - Как и в консоли для вызова команды нужно ввести её имя и аргументы например:
        ls -r -l
        ls -l file1 file2 folder1
        т.д.
    - consol_myls.exe - исполняемый файлы (скомпилируемая программа)
    - consol_myls_vr.c - исходный код программы
    - Сборка производилась на gcc (Ubuntu 13.3.0-6ubuntu2~24.04.1) 13.3.0
    - программа работает только на Linux!!

### Myls
    - Скрипт, который выполняет вывод информации о файле подобно команде ls
    - Скрипт вызывается из консоли в него передаюся нужные аргументы например:
        ./myls.exe -r -l                // где ./myls.exe - имя скрипта
        ./myls.exe -l file1 file2 folder1
        т.д.
    - myls.exe - исполняемый файлы (скомпилируемая программа)
    - myls_vr.c - исходный код программы
    - Сборка производилась на gcc (Ubuntu 13.3.0-6ubuntu2~24.04.1) 13.3.0
    - программа работает только на Linux!!
